# Manager-System
信息管理系统，可用于学生，人事等管理，SSM框架
数据库：Oracle数据库 需要修改jdbc.propertites及相关数据库配置文件
开发工具：Eclipse
页面：jsp采用Extjs编写
该系统具有基本的登录，增删改查功能，自己也可添加需要的页面和业务。
